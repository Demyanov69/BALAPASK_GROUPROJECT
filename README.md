# Проект: игра «Ним»

## Предисловие

**«Ним»** — математическая комбинаторная игра, в которой два игрока по очереди берут предметы из одной или нескольких кучек.  

В нашем проекте реализована **одна кучка**, из которой игрок и бот берут 1–3 спички за ход. Побеждает тот, кто **забирает последнюю спичку**.

## Идея проекта

Создать интерактивную консольную игру Ним на Prolog с ботом, выступающим как второй игрок (противник). Бот умеет объяснять свои ходы и использует математическую стратегию.

## Как играть

1. Перейдите в папку проекта:

```bash
cd C:/Users/YourName/Desktop/nim_bot
```

2. Запустите SWI-Prolog:

```bash
swipl
```

3. Установите кодировку UTF-8 (для русского языка):

```prolog
:- set_prolog_flag(encoding, utf8).
```

4. Загрузите главный файл:

```prolog
?- [main].
```

5. Запустите игру:

```prolog
?- main.
```

6. Введите количество спичек, и поочередно берите 1–3 спички.  

### Пример запуска

```prolog
Введите количество спичек для игры: 21

Добро пожаловать в игру Nim!
Всего спичек: 21
| | | | | | | | | | | | | | | | | | | | |
Вы ходите первым.
--------------------------
Осталось спичек: 21
| | | | | | | | | | | | | | | | | | | | |
Сколько спичек взять (1-3)? |:
```
